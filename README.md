# CryptoCatnip
Tamper-evident image verification system using blockchain and visual indicators to detect and highlight media manipulation.

## 🚀 Features
- Image integrity verification using hashing
- Tamper detection system
- Visual distortion for tampered images
- Blockchain-based integrity concept

## 🛠 Tech Stack
- Python (Flask)
- PIL (Image Processing)
- HTML/CSS (UI)

## ▶️ How to Run

```bash
git clone https://github.com/YOUR_USERNAME/CryptoCatnip.git
cd CryptoCatnip
pip install -r requirements.txt
python app.py
```
